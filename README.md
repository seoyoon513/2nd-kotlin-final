### 기능 요구사항

---
[구현]
- `+ - / *` 사칙연산 버튼 기능 구현
- `C` 버튼 기능 구현
- `AC` 버튼 기능 구현
- `√` ****루트 버튼 ****기능 구현
- `▶` 지우기 버튼 기능
- `GT` 기능 구현
- `.` 소수점 입력 기능 구현
- 라운딩 셀렉터 구현
    - F → 소수점 이하 수를 있는 그대로 표시, 14자리에 맞추어서 출력
-자리수 셀렉터 구현
    - ADD2 → 모든 수를 기본적으로 소수점 2자리 수로 변환
    
- 기능구현 제외 버튼 `H/M/S`, `%`
    - 클릭시 ‘기능 구현 예정입니다’ 메세지 표시

[미구현]
- `+/-` 버튼 기능 구현
- `K` 기능 구현
- `MC`, `MR`, `M-`, `M+`  기능 구현
- 1,000단위 기준 , 표시
- 라운딩 셀렉터 구현
    - CUT → 버림
    - 5/4 → 반올림
- 자리수 셀렉터 구현
    - 소수점 자리수 4, 3, 2, 1, 0 표시



### 예외처리 고려사항
---
- 소수점 중복 입력 방지 처리
- 계산 결과가 ```.0```으로 끝날 경우 ```.0```생략
- 0일 때 0, 00
